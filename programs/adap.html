Course Descriptions

1. Statistics for Data Analysis

+
Short Modular (Core Theory) 3 Weeks (Month 1)

Topics Covered:

  • Descriptive statistics and data visualization
  • Probability theory and distributions
  • Hypothesis testing and confidence intervals
  • Statistical inference and interpretation

2. Data Cleaning and Preparation with Excel

+
Short Modular 3 Weeks (Month 1)

Topics Covered:

  • Handling missing data and duplicates
  • Data formatting and standardization
  • Data readiness and validation
  • Excel functions for data preparation

3. Excel for Data Analysis

+
Core Skill Across 4 Months

Topics Covered:

  • Advanced Excel functions and formulas
  • Pivot tables and data summarization
  • Data analysis workflows in Excel
  • Reporting and visualization in Excel

4. Exploratory Data Analysis with Python

+
Advanced Tool 4 Weeks (Month 3)

Topics Covered:

  • Pandas for data manipulation
  • NumPy for numerical operations
  • Matplotlib for data visualization
  • EDA techniques and best practices

5. Power BI for Data Visualization

+
Advanced Tool 2 Weeks (Month 4)

Topics Covered:

  • Creating interactive dashboards
  • Business reporting and KPIs
  • Data modeling in Power BI
  • Dashboard publishing and sharing

6. Tableau for Dashboard Design

+
Advanced Tool 5 Weeks (Month 3–4)

Topics Covered:

  • Data storytelling principles
  • Dashboard design and layout
  • Interactive visualizations
  • Dashboard presentation skills

7. Cross-Program Seminars

+
Seminars Invited Professionals

Topics Covered:

  • Digital & Data Thinking (1 Day)
  • Responsible Use of AI (1 Day)
  • Data Visualization & Interpretation (2 Days)
